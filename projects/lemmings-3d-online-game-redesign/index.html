Lemmings 3D Online - Game Redesign

MultiplayerNetworkingUnity

Lemmings 3D is a project I developed with two colleagues for a network and multiplayer class on my digital games development graduation.

The project reimagines the classic Lemmings game as an online 3D versus game. We implemented the network system allowing two people to compete each other via the internet. Lemmings from opposing teams can't interact themselves, but their actions on obliterating walls and floors are shared across clients.
