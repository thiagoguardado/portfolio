Ant Fall - GBJAM5

BrowserGame BoyGame JamUnity

This little project was made for submission on GBJAM5 (GameBoy Jam 5), hosted by itch.io.

The objective of the game is to travel the farthest possible bouncing over floating leafs, using arrow keys to move the ant sideways and space bar to open its leaf-parachute and float (falling slower).

All sprites and sounds are original and it was set up using Unity Engine.
