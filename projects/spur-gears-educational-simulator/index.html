Spur Gears Educational Simulator

FrontendHTML5PixiJSSimulator

One of the projects I developed working for SENAI-RS was a spur gears simulator, aimed to help students consolidate their learning on the transmission relations between spur gears.

The project was implemented in HTML5/Javascript/CSS using PixiJs V5. It was made as a PWA (Progressive Web-App), which means it can be accessed through desktop and mobile browsers and installed on the device to be used offline. You can access the simulator using the link below:

spur gears simulator

Using Electron, the simulator was also distributed as an executable for Windows for teachers and students across the state.
